What the Project is All About

Reservoir petrophysical evaluation is a critical aspect of petroleum engineering that involves determining key formation properties such as porosity, permeability, and fluid saturation. These properties help engineers estimate hydrocarbon reserves and evaluate the productivity potential of an oil or gas reservoir.

Well logging provides continuous measurements of subsurface formations, and the data obtained are commonly stored in LAS (Log ASCII Standard) format. Traditionally, the interpretation of well log data and computation of petrophysical parameters are performed manually or using expensive commercial software.

This project focuses on developing a computer-based web application using the Django framework to automate the evaluation of reservoir petrophysical properties. The system will allow users to upload LAS files, process the well log data, and compute important parameters such as porosity, water saturation (using Archieâ€™s equation), and permeability (using empirical correlations).

The developed application aims to improve computational efficiency, reduce human error, and provide an accessible and cost-effective tool for academic and basic reservoir evaluation purposes. It also enhances the understanding of well log interpretation and digital reservoir characterization.
